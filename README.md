❄️ NixOS Configuration (vijetasatyam)

This repository contains my personal, modular NixOS configuration. It is built to be fully reproducible, meaning this entire system can be rebuilt from scratch on any hardware using only these files.
🏗️ Architecture: Folder-Wise Hierarchy

This repository follows a professional modular structure. Logic is separated by concern to ensure the system remains maintainable as it grows.
📂 Directory Structure

    hosts/nixos/: Machine-specific settings. Contains the main configuration.nix and hardware-configuration.nix.

    modules/core/: The foundation. Includes system.nix, networking.nix, users.nix, services.nix, and aliases.nix.

    modules/apps/: Software manifests for both stable and unstable channels.

    modules/dev/: Development environment, including git-config.nix and GPG setups.

    scripts/: Helper scripts for system maintenance.

🚀 Management & Maintenance

Custom shell aliases are defined in modules/core/aliases.nix to streamline daily operations:
Action	Alias	Command Equivalent
Apply Changes	rebuild	sudo nixos-rebuild switch
Full Upgrade	upgrade	sudo nix-channel --update && sudo nixos-rebuild switch
Emergency Fix	rollback	sudo nixos-rebuild switch --rollback
Deep Clean	nix-clean	sudo nix-collect-garbage -d && sudo nix-store --optimise
Dual Sync	sync	git push origin main && git push github main
Navigate	conf	cd ~/nixos-config

🛠️ Installation on a New Machine

    Clone the repo:
    Bash

git clone git@codeberg.org:vijetasatyam/nixos-config.git ~/nixos-config

Hardware Configuration: Copy the hardware-configuration.nix generated by the installer into ~/nixos-config/hosts/nixos/.

Link and Build:
Bash

    sudo ln -sf ~/nixos-config/hosts/nixos/configuration.nix /etc/nixos/configuration.nix
    sudo nixos-rebuild switch
